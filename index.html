Reference

USML (pronounced ooze-mel) is a small language intended for writing tiny programs based around string manipulation. The language is still very much in its infancy.

Commands

  • h (Hello, world!)

    The h command adds the string "Hello, world!" to the output. That's it.

  • " (Begin/End String)

    The " command begins a string. If the interpreter reaches the end of the program before the string is ended, the ending is not required.

    Escape Codes

    • \n: newline
    • \t: tab

    You can also put a backslash before any character not listed above and it will be interpreted as that character.

    Examples

    A program whose output is identical to the h command:

    "Hello, world!

  • ' (Begin/End String with Space)

    If used to begin a string, this command creates a string with a space. If used to end a string, it inserts a space before ending the string.

  • u (Uppercase)

    Converts the next string to uppercase.

  • l (Lowercase)

    Converts the next string to lowercase.

  • s (Substring One)

    Gets a substring of a string.

    Examples

    Gets the characters after the first two.

    s2"This is a string.

  • S (Substring Two)

    Gets a substring of a string.

    Examples

    Gets the first four characters of a string.

    S0"This is a string."4

  • i (Input)

    Returns the input provided to the program.

  • q (Quine)

    Returns the program's source code.

  • E (End Execution)

    Stops execution of the program.

  • c (Get Character)

    Returns the next character in the program.
